# 우리 반 교실놀이 게시판

앞으로 만들어갈 다양한 교실놀이(수업용 게임)들을 하나씩 추가해 나갈 웹페이지입니다. 새로운 놀이가 만들어질 때마다 이 게시판에 카드 형태로 등록되어, 학생들이 목록에서 골라 바로 실행할 수 있도록 구성됩니다.

게시판에 새로운 놀이를 **추가**하는 기능은 관리자(교사) 본인의 구글 계정으로 로그인했을 때만 사용할 수 있습니다. 로그인하지 않은 일반 방문자는 등록된 놀이 목록을 보고 실행하는 것만 가능합니다.

## 현재 등록된 게임

### 스트림스 (Streams)

수업 시간(주로 수학) 중 짧게 진행할 수 있는 보드게임 '스트림스'를 웹으로 구현한 실시간 참여형 게임입니다.

**진행 방식**

1. 교사(방장)가 방을 생성하면 4자리 방 코드가 큰 글씨로 표시됩니다.
2. 학생(참가자)은 태블릿 등에서 방 코드를 입력하고 닉네임을 등록해 방에 입장합니다.
3. 교사가 참가자 목록을 확인한 뒤 게임을 시작하면, "뽑기" 버튼을 눌러 숫자 타일을 하나씩 무작위로 뽑습니다.
4. 학생은 뽑힌 숫자를 자신의 20칸 판 중 빈 칸에 배치합니다. 모든 학생이 배치를 완료해야 교사가 다음 숫자를 뽑을 수 있습니다.
5. 한 번 배치한 숫자는 이동할 수 없으며, 20칸이 모두 채워지면 게임이 종료됩니다.
6. 게임이 끝나면 각자의 판을 구간별로 채점하여 전체 순위표(닉네임+점수)가 공개됩니다.

게임 중에는 다른 학생의 판과 점수가 서로에게 보이지 않으며, 게임 종료 후에만 전체 결과가 공개됩니다.
